About This Repository

This repository is a direct fork of the official Keiyoushi Extensions Source repository.

I've created this fork primarily for my personal backup, to ensure I always have access to these extensions. While it's available publicly, its main purpose is for my own use. You can keep your fork up-to-date by regularly syncing your fork.

Why Trust This Fork?

The extensions in this repository retain the **exact same signing key fingerprint** as the original official extensions. This means Komikku verifies them as legitimate updates from a trusted source.
